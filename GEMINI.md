# Dimmed Monokai Theme

This repository contains a collection of configuration files for a "Dimmed Monokai" theme, designed for a Linux desktop environment. The theme uses a dark background with green and pink/purple accents, and is available for a variety of applications.

## Directory Overview

The repository contains the following files:

*   **`alacritty.toml`**: Configuration for the Alacritty terminal emulator.
*   **`btop.theme`**: Theme for the btop system monitor.
*   **`chromium.theme`**: Theme for the Chromium web browser.
*   **`ghostty.conf`**: Configuration for the Ghostty terminal emulator.
*   **`hyprland.conf`**: Configuration for the Hyprland window manager.
*   **`hyprlock.conf`**: Configuration for the Hyprlock screen locker.
*   **`icons.theme`**: Icon theme.
*   **`kitty.conf`**: Configuration for the Kitty terminal emulator.
*   **`mako.ini`**: Configuration for the Mako notification daemon.
*   **`neovim.lua`**: Configuration for the Neovim text editor.
*   **`swayosd.css`**: Configuration for the Swayosd on-screen display.
*   **`walker.css`**: Configuration for the Walker application launcher.
*   **`waybar.css`**: Configuration for the Waybar status bar.
*   **`wofi.css`**: Configuration for the Wofi application launcher.
*   **`backgrounds/`**: A directory containing desktop backgrounds.

## Usage

To use this theme, you will need to copy the relevant configuration files to your system's configuration directories. For example, to use the Hyprland theme, you would copy `hyprland.conf` to `~/.config/hypr/hyprland.conf`.

Please refer to the documentation for each individual application for more information on how to apply the themes.
